# Symfony Docker

A [Docker](https://www.docker.com/) based installer for the [Symfony](https://symfony.com) web framework. 

## Getting Started
````
make build
make up
````

## Installation
````
make install
````

## Usage
````
Open `http://127.0.0.1` in your favorite web browser
````

## Debugging
````
docker-compose logs -f php-fpm
````
